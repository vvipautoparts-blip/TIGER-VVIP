## 📋 الخطوة 1: الحصول على بيانات Resend (5 دقائق)

### أ. إنشاء حساب Resend
```
1. توجه إلى: https://resend.com
2. اضغط: "Get Started"
3. أكمل التسجيل (Email + Password)
4. تحقق من بريدك الإلكتروني
```

### ب. الحصول على API Key
```
1. بعد الدخول، اذهب إلى: API Keys
2. اضغط: "Create API Key"
3. اختر: "Full Access"
4. انسخ المفتاح الظاهر (لن يظهر مرة أخرى!)
5. احتفظ به آمناً
```

**مثال:**
```
API Key: re_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### ج. توثيق Sender Domain (اختياري)
```
إذا كان لديك دومين:
1. في Resend Dashboard → Domains
2. أضف دومينك
3. أتبع خطوات التحقق (DNS)
4. استخدم: noreply@yourdomain.com

إذا لم يكن لديك دومين:
- استخدم: onboarding@resend.dev (مؤقتاً)
```

## 🔧 الخطوة 2: إعداد SMTP في Supabase (10 دقائق)

### أ. افتح Supabase Dashboard
```
1. توجه إلى: https://app.supabase.com
2. اختر مشروع: TIGER VVIP
3. اذهب إلى: Project Settings (أيقونة الترس بالأسفل)
```

### ب. انتقل إلى Email Settings
```
1. في الشريط الجانبي → Authentication
2. أسفل: Email (Confirm Signup)
3. ابحث عن: SMTP Provider
4. اختر: Custom SMTP
```

### ج. ملء بيانات SMTP
```
┌─────────────────────────────────────────┐
│ SMTP Configuration                      │
├─────────────────────────────────────────┤
│ SMTP Host:        smtp.resend.com       │
│ SMTP Port:        587                   │
│ SMTP User:        resend                │
│ SMTP Password:    [API Key من Resend]   │
│ Sender Email:     noreply@resend.dev    │
│ Sender Name:      TIGER-VVIP           │
└─────────────────────────────────────────┘
```

**انقر:** Save

## 📧 الخطوة 3: تطبيق قوالب البريد الاحترافية (15 دقائق)

### أ. اختر نوع البريد
```
في Supabase → Authentication → Email Templates

اختر: "Confirm signup" (هذا هو نوع البريد الذي نريده)
```

### ب. نسخ القالب الاحترافي

**للنسخة العربية:**

1. افتح: `/workspaces/TIGER-VVIP/email-templates/confirm-email.html`
2. انسخ كل المحتوى
3. في Supabase → "Confirm signup" → HTML
4. امسح المحتوى القديم
5. الصق القالب الجديد
6. اضغط: Save

**للنسخة الإنجليزية:**

1. افتح: `/workspaces/TIGER-VVIP/email-templates/confirm-email-en.html`
2. انسخ كل المحتوى
3. في نفس صفحة Supabase
4. امسح المحتوى
5. الصق القالب الإنجليزي
6. اضغط: Save

### ج. تعيين URL إعادة التوجيه

**في نفس صفحة "Confirm signup":**

```
Find:    Redirect URL
Set to:  https://vvipautoparts-blip.github.io/TIGER-VVIP/#registration-page?verified=true
```

⚠️ **مهم:** تأكد من أن الرابط ينتهي بـ `#registration-page?verified=true`

## ✅ الخطوة 4: اختبار النظام (5-10 دقائق)

### أ. اختبار في البيئة المحلية
```
1. شغّل: python -m http.server 8000
2. اذهب إلى: http://localhost:8000/#registration-page
3. أدخل بريدك الاختباري
4. اضغط: "Verify Email"
5. ✅ تحقق من أن الرسالة وصلت
```

### ب. اختبر الرابط في الرسالة
```
1. افتح الرسالة المستقبلة
2. اضغط: "تفعيل الحساب الآن"
3. ✅ يجب أن تنتقل لصفحة التحقق
4. ✅ يجب أن تظهر رسالة نجاح
5. ✅ يجب أن تنتقل تلقائياً لصفحة البروفايل
```

### ج. اختبار في Production (GitHub Pages)
```
1. اذهب إلى: https://vvipautoparts-blip.github.io/TIGER-VVIP/#registration-page
2. أدخل بريدك الحقيقي
3. تحقق من البريد
4. ✅ يجب أن تعمل بالكامل
```

## 🚨 الأخطاء الشائعة والحلول

### المشكلة: "Emails not sending"

**الحل:**
```
✅ تحقق من API Key (انسخه بحذر)
✅ تأكد من أن Port = 587 (ليس 25 أو 465)
✅ اختبر في Resend Dashboard أولاً
✅ تحقق من Sender Email المعتمد
```

### المشكلة: "رابط التفعيل غير صحيح"

**الحل:**
```
✅ تأكد من URL في "Redirect URL"
✅ تأكد من أنه ينتهي بـ #registration-page
✅ أضف ?verified=true في النهاية
✅ لا تستخدم مسافات في الرابط
```

### المشكلة: "القالب لا يظهر بشكل صحيح"

**الحل:**
```
✅ انسخ القالب كاملاً (بدون تعديلات)
✅ تأكد من أنك في صفحة "Confirm signup"
✅ امسح الـ Cache في متصفحك
✅ أعد تحميل صفحة Supabase
```
