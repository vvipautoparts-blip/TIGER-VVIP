# TIGER-VVIP

مشروع واجهة أولية لموقع قطع غيار السيارات الفاخرة TIGER VVIP.
المحتوى يدعم العرض باللغة العربية والإنجليزية مع كتالوج تفاعلي وخيارات بحث.

## الملفات الرئيسية

- `index.html` : الصفحة الرئيسية للموقع.
- `styles.css` : تصميم واجهة المستخدم.
- `script.js` : تفاعل البحث وتبديل اللغة، ونموذج الطلب ولوحة الموافقة.

## المزايا الجديدة

- نموذج طلب قطع يعمل باللغة العربية والإنجليزية.
- لوحة متابعة الطلبات مع حالة الطلب: "قيد الانتظار"، "موافق"، "مرفوض".
- دعم تسجيل دخول وتسجيل مستخدمين عبر Supabase.
- دعم دور المستخدم `dealer` و`admin` في التسجيل.
- صفحة إدارة الموردين مع حفظ بياناتهم في Supabase، ونموذج الموردين متاح للمشرفين فقط.
- تخزين الطلبات في جدول `orders` حقيقي في Supabase.

## إعداد Supabase

1. أنشئ مشروع جديد في Supabase.
2. انسخ قيمة `SUPABASE_URL` و`SUPABASE_ANON_KEY` من إعدادات المشروع.
3. افتح `supabase-config.js`، واستبدل القيم التالية بقيمك:
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`
4. في لوحة Supabase SQL، نفّذ محتوى الملف `supabase-schema.sql` لإنشاء الجداول.
5. تأكد من تفعيل مصادقة البريد الإلكتروني في Supabase Authentication.
6. فعّل موفّري Google وFacebook في Supabase Auth إذا أردت تسجيل الدخول الاجتماعي.
7. أضف Redirect URL الخاص بمشروعك إذا كنت تختبر محلياً، مثلاً:
   - `http://localhost:8000`
   - أو عنوان النطاق الخاص بالموقع عند النشر

## الحقول الجديدة في قاعدة البيانات
- جدول `profiles` يحتوي الآن على حقل `subscription` لحفظ خطة الاشتراك.

## تشغيل المشروع

1. افتح `index.html` في المتصفح مباشرة.
2. أو استخدم خادم محلي بسيط مثل:
   - `python3 -m http.server 8000`
   - ثم افتح `http://localhost:8000`
