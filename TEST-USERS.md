## 📋 خطوات الاختبار

### الخطوة 1️⃣: اختبار صفحة الدخول

```
1. افتح: http://localhost:8000
2. انقر على "تسجيل الدخول"
3. أدخل البيانات:
   - البريد: admin@tigervvip.com
   - كلمة المرور: Password123!
4. انقر "دخول"
5. ✅ يجب أن تنتقل للبروفايل تلقائياً (8 ثوان)
```

### الخطوة 2️⃣: اختبار صفحة البروفايل

```
✅ التحقق من العناصر:
├─ صورة الغلاف (#e4e6eb)
├─ صورة البروفايل الدائرية (168px)
├─ اسم المستخدم (28px، Bold)
├─ نوع الحساب ("المدير العام")
├─ رقم الهاتف
├─ البريد الإلكتروني
├─ عدد الطلبات
├─ تاريخ الانضمام
└─ زر "تعديل البروفايل" (أزرق فيسبوك)
```

### الخطوة 3️⃣: اختبار صفحة Home Feed

```
1. من البروفايل، انقر "Home"
2. ✅ يجب أن تظهر صفحة Home بـ 3 أعمدة:
   ├─ اليسار: الفئات والتنقل
   ├─ الوسط: بطاقات الخدمات
   └─ اليمين: روابط سريعة
3. اختبر الـ Dropdown (تصفية حسب الفئة)
4. اختبر الأزرار (اتصال + واتساب)
```

### الخطوة 4️⃣: اختبار أزرار الاتصال

```
✅ زر الاتصال:
└─ انقر → يفتح tel:+962780003302

✅ زر الواتساب:
└─ انقر → يفتح wa.me/962780003302

✅ معلومات الاتصال:
├─ الهاتف: +962780003302
├─ البريد: info@tigervvip.com
├─ العنوان: عمان - العبدلي
```
