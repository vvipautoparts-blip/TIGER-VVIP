# TIGER AutoParts JO AI Prompts

## 1) FlutterFlow Prompt (Arabic)

ابنِ تطبيق FlutterFlow باسم TIGER AutoParts JO لسوق قطع غيار السيارات ومراكز الصيانة في الأردن.

المتطلبات:
- تصميم VVIP داكن بالأسود/الكحلي ولمسات ذهبية.
- دعم كامل للعربية RTL والإنجليزية LTR مع تبديل لغة سهل.
- أربعة أدوار: Super Admin، Field Representative، Merchant/Service Center، Customer، بالإضافة إلى وضع زائر Guest للعرض فقط.
- Firebase مع مجموعات: Users، Merchants، ServiceCenters، Companies، Services، Parts، Vehicles، Categories، Subscriptions، Approvals، Notifications.
- تسجيل برقم الهاتف مع OTP واستعادة كلمة المرور عبر الهاتف.
- سير موافقة Pending / Approved / Rejected. لا يظهر للمستخدمين إلا المحتوى المعتمد.
- بحث وفلاتر متقدمة حسب الماركة، الموديل، السنة، نوع الهيكل، الفئة، الموقع، والسعر.
- لوحات تحكم منفصلة لكل دور.
- صفحات: Splash، Login/Register، Home/Guest، Parts Catalog، Part Details، Stores/Centers List، Center Details، Services، Merchant Dashboard، Representative Dashboard، Admin Dashboard، Approval Center، Settings.

## 2) Wix Prompt (English)

Create a premium bilingual Wix site for TIGER AutoParts JO in Jordan. Use a dark luxury VVIP theme with black/navy and gold accents.

Requirements:
- Full Arabic RTL and English LTR support with easy language switching.
- Role-based access for Super Admin, Field Representative, Merchant/Service Center, Customer, plus Guest browse mode.
- CMS collections for Users, Merchants, ServiceCenters, Companies, Services, Parts, Vehicles, Categories, Subscriptions, Approvals, Notifications.
- Phone login with OTP or password reset via phone.
- Approval workflow Pending / Approved / Rejected; only approved items visible to customers.
- Advanced filters for make, model, year, body type, category, location, and price.
- Separate pages/dashboards for each role.
- Pages: Home, Parts Catalog, Part Details, Stores/Centers List, Center Details, Services, Login/Register, Merchant Dashboard, Representative Dashboard, Admin Dashboard, Approval Center, Settings.
